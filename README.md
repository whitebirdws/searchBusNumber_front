## □.프로젝트 명칭 및 소개

- 프로젝트명: searchBusNumber_front
- 경기도 버스노선 조회 서비스를 구현한 토이프로젝트 입니다.

## □.프로젝트 목적

- openAPI를 활용하여 경기도 버스번호를 검색 하여 그에 해당하는 버스노선을 시각적으로 출력
- 출력된 버스 노선에서 선택한 정류장의 도착 정보를 확인할 수 있는 서비스를 목표로 제작
